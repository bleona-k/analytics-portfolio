# US Consumption Regression & Forecasting

This project analyzes the relationship between U.S. consumption and income using time series regression models.

## Methods Used
- Simple Linear Regression (TSLM)
- Multiple Regression
- Residual diagnostics
- Forecast evaluation

## Tools
- R
- fable / tsibble
- Time series modeling
